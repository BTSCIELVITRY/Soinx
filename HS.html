Bienvenue dans le monde du HTML et du CSS

Apprenez les bases pour concevoir vos propres sites web !

Qu'est-ce que le HTML ?

HTML (HyperText Markup Language) est le langage de balisage utilisé pour structurer le contenu d'une page web. Voici un exemple de balisage HTML : 

<h1>Ceci est un titre</h1>
<p>Ceci est un paragraphe.</p>

Qu'est-ce que le CSS ?

CSS (Cascading Style Sheets) est utilisé pour styliser les pages HTML, c'est-à-dire ajouter des couleurs, modifier les polices, gérer les espaces, et bien plus encore. 

h1 {
    color: blue;
    font-size: 2em;
}

Comparaison entre HTML et CSS

HTML CSS
Structure le contenu Ajoute du style
Utilise des balises comme <h1>, <p>, <div> Utilise des sélecteurs comme h1, .classe, #id
Ne contrôle pas l'apparence Ne contrôle pas le contenu
